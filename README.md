# Proyecto de Conexión: Node.js + Supabase (FIME)

Este proyecto forma parte de las actividades de laboratorio para la facultad de **FIME**. Consiste en una aplicación básica que conecta un entorno local con una base de datos en la nube utilizando **Supabase**.

## 🚀 Funcionalidades
- **Conexión al Backend**: Configuración de cliente Supabase en Node.js.
- **Base de Datos Relacional**: Creación y gestión de una tabla `estudiantes`.
- **Inserción de Datos**: Scripts para cargar registros de forma automatizada y manual.
- **Interfaz Web**: Formulario HTML para captura de datos del usuario.
- **Seguridad y Auth**: Implementación de Autenticación básica y manejo de políticas RLS.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje**: JavaScript (Node.js).
- **Base de Datos**: Supabase (PostgreSQL).
- **Control de Versiones**: Git y GitHub.
